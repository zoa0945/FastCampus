# SpotifyStyleLoginApp

Firebase 연결 및 로그인 화면 구현 공부를 위해 만든 샘플앱입니다.

## Introduction
Firebase 인증 서비스를 통해 다양한 방법으로 로그인 할 수 있는 기능 구현
1. 이메일 / 비밀번호로 로그인
2. Google 계정으로 로그인

## Development Environmnet
* Swift
* Cocoapods
* Firebase
