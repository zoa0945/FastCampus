# NetflixStyleSampleApp
snapKit을 이용한 UI구성과 UICollectionView 활용 연습을 위한 샘플앱입니다.

## Introduction
storyboard 없이 UI를 그리고 collectionView의 CompositionalLayout 구성
1. snapKit을 활용하여 오브젝트 생성 및 오토 레이아웃 설정
2. 각 섹션별 CompositionalLayout 구현
